# Values in Nursing

## Definition
**Values in nursing** are the beliefs and ideals that influence how nurses think, act, and care for patients.

## Key Points
- Values guide nurses in daily practice.
- They help nurses make good decisions.
- Important nursing values include honesty, respect, caring, and responsibility.

## Detailed Explanation

Values in nursing are important because they:
1. Build trust between nurse and patient.
2. Improve quality of care.
3. Guide ethical behavior.
4. Support teamwork.
5. Help nurses face difficult situations.

Core nursing values:
- **Patient-centered care** – Putting the patient first.
- **Respect** – Honoring patient dignity and choices.
- **Compassion** – Showing kindness and understanding.
- **Integrity** – Being honest and ethical.
- **Excellence** – Striving for the best care.

## Bullet Summary
- Values = beliefs that guide nursing actions
- Important values: respect, compassion, integrity, excellence, patient-centered care
- Values improve trust, care quality, and ethical practice.
